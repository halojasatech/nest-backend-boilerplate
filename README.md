## Description

[Nest](https://github.com/nestjs/nest) complete boilerplate repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# run with docker for local
$ docker-compose up

# debug mode
$ npm run start:debug

# development mode
$ npm run start

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
